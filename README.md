### Usage
點擊數字鍵輸入數值。

點擊運算符號選擇運算方法，並清空顯示數值。

再次點擊數字鍵輸入數值並點擊等於鍵即可顯示計算結果。

點擊 C 清空顯示數值。
數值紀錄與運算功能。含數字鍵、四則運算鍵、等於鍵以及歸零鍵。

以布林值控制計算階段與防止小數點疊加，用String取得算式結果作為一個字串存取。
